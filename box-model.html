Box model

Everything displayed by CSS is a box., and understanding these boxes is key to being able to create layouts with CSS, or to align items with other items.

The areas of the box model

content box

This is the areas where the content, forexample, text, video, image e.t.c live

It is the inner most area of box model. The content determines the content area size

padding box

This is the area that surrounds the content area

We can manage this space using the padding css property. It is possible to define individual sides, that is to say, top, right, bottom and left. Or we can set all at once

Padding property values can be defined in length, forexample, px, pt, cm, etc., OR in percentage % - which specifies a padding in % of the width of the containing element

border box

The border box surrounds the padding box and its space is occupied by the border value. The border box is the bounds of your box and the border edge is the limit of what you can visually see. The border property is used to visually frame an element.

Margin box

The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

Block and inline boxes

In CSS we broadly have two types of boxes â€” block boxes and inline boxes. These characteristics refer to how the box behaves in terms of page flow and in relation to other boxes on the page. Boxes also have an inner display type and an outer display type. First, we will explain what we mean by block box and inline box. We will then explain what is meant by an inner and outer display type.

If a box has an outer display type of block, it will behave in the following ways:

  • The box will break onto a new line.
  • The box will extend in the inline direction to fill the space available in its container. In most cases this means that the box will become as wide as its container, filling up 100% of the space available.
  • The width and height properties are respected.
  • Padding, margin and border will cause other elements to be pushed away from the box

HTML elements like - Headings, Divs, Paragraph, Section, Ol, ul, Table. By default they use block display type

If a box has an outer display type of inline, then:

  • The box will not break onto a new line.
  • The width and height properties will not apply.
  • Vertical padding, margins, and borders will apply but will not cause other inline boxes to move away from the box.
  • Horizontal padding, margins, and borders will apply and will cause other inline boxes to move away from the box.

Some HTML elements, such as anchor tags, span, em and strong use inline as their outer display type by default.
